Mini Golf Generator


Origins


I was looking through the assets that http://kenney.nl has released for free use and found a package of mini golf assets, and decided to start up a short little project.

I've always wanted to try and create a randomizer that creates paths and this is what I've come up with so far.

The current issue is the lack of an end goal to this game, as there are hundreds of mini golf games out there and I don't want to release a generic mini golf game
